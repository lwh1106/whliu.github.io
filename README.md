# Weihao Liu Personal Homepage

Static personal homepage prepared for GitHub Pages.

This is a simplified, single-page version inspired by the AcademicPages layout:
top navigation, author sidebar, and compact academic sections. It does not require
Jekyll, Ruby, or bundled theme dependencies.

## Publish on GitHub Pages

1. Push this folder to a GitHub repository.
2. In the repository settings, enable GitHub Pages.
3. Select the branch and root folder that contain `index.html`.

The site is self-contained: `index.html`, `assets/styles.css`, the CV PDF, and
local research figures are all included.
